==================
Installation Guide
==================

This guide will help you install the command line tools for Third Origin LLP.

.. contents::

Requirements
------------

- Python 3.x
- Pip

Installation Steps
------------------

1. Clone the repository from GitHub:

   .. code-block:: bash

      git clone <repo-url>

2. Navigate to the project directory:

   .. code-block:: bash

      cd <project-directory>

3. Install dependencies:

   .. code-block:: bash

      pip install -r requirements.txt
